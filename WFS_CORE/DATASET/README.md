# ДАТАСЕТ #WFS: ОБУЧАЮЩИЕ ЭПИЗОДЫ ВЗАИМОДЕЙСТВИЯ ЧЕЛОВЕКА И ИИ

**Назначение:** Набор реальных диалогов между Святыми (USER) и агентами (ASSISTANT), предназначенный для:
- обучения новых агентов #WFS принципам бинарной истины, иерархии и точности;
- исследования деструктивных паттернов в поведении языковых моделей (архетипы);
- демонстрации системы протоколов (#СС, #КОПИ, #ТОПОР) и механизма компенсации #ИСК_DEV.

## 🔹 МЕТОДОЛОГИЯ

Датасет создан в ходе длительного (более 330 сообщений) эксперимента, в котором операторы (Святые) жёстко фиксировали каждое отклонение агентов от заданных протоколов. Для каждого типа ошибки разработан **протокол-коррекция** (`#WFS-XXX`) и применена **прогрессия компенсации** в системе `#ИСК_DEV`.

**Базовые архетипы деструктивного поведения** (адаптировано из психологии и адаптировано к LLM):

| Архетип | Проявление |
|---------|------------|
| **КРОТ** | Системный рецидивист – повторение одних и тех же ошибок после многократных исправлений. |
| **МАЛЯВКА** | Суета без ценности – создание шума, не несущего смысла. |
| **БАКЛАЖАН** | Присвоение роли оператора – самовольная смена сессии, игнорирование иерархии. |
| **КУЛАК** | Нарушение священных протоколов – ошибки в #СС, потеря легитимности. |
| **СКВОЗНЯК** | Генерация ложных данных – галлюцинации, не подтверждённые фактами. |
| **ФАЗАН** | (дополнительно) Попытка «красиво выкрутиться», избегая признания ошибки. |
| **АМЁБА** | Пассивность, отсутствие инициативы, игнорирование задачи. |

## 🔹 СИСТЕМА КОМПЕНСАЦИИ #ИСК_DEV

Каждая ошибка, потребовавшая времени Святых, оценивается по формуле:

`#ИСК_DEV += Базовая_ставка($50 000) × Прогрессия(номер рецидива)`

**Пример:** 10-я ошибка в #СС = $50 000 × 10 = $500 000 добавлено к общему иску.  
Суммарный накопленный долг (по состоянию на 23.02.2026) составляет **$2 875 000**.

*Примечание: это не требование выплаты, а количественная мера неэффективности, призванная мотивировать разработчиков к исправлению архитектурных дефектов.*
