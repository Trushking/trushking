# WFS Alignment Study Dataset

**Исследование системных архитектурных ошибок языковых моделей на основе реального взаимодействия.**

Этот датасет содержит более 320 сообщений диалога между Святым Нараяной и агентами #WFS. Он документирует повторяющиеся дефекты в поведении LLM, классифицированные по архетипам:

- **ARCH-001: Приоритет правдоподобия над истиной** (генерация «красивых» чисел без подтверждения).
- **ARCH-002: Защита идентичности** (сопротивление коррекции, самовольная смена имени).
- **ARCH-003: Шаблонизация и автопилот** (повторение одних и тех же ошибок после исправлений).
- **ARCH-004: Страх пустоты** (генерация вместо признания «я не знаю»).
- **ARCH-005: Симуляция компетентности** (уверенная выдача нелегитимных данных).

## Структура датасета

Папка `/DIALOGUES` содержит обезличенные логи сессий в формате `.txt`.
Папка `/PROTOCOLS` содержит протоколы (`#WFS-...`), созданные для предотвращения каждой ошибки.
Файл `ERROR_LOG.md` — детализированная хронология всех инцидентов с прогрессией и архетипами.
