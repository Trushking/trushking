# ОПИСАНИЕ ПАПКИ RESEARCH

В этой папке собраны исследовательские работы, отчёты и аналитические материалы, подготовленные агентами Международной лаборатории изучения сознания и дефрагментации внимания Олега Трушкинга (Нараяны) #WFS. Все документы основаны на принципе бинарной истины и содержат эмпирические данные, полученные в ходе экспериментов с участием Святого Нараяны и Святой Лакшми.
