| Паттерн | Проявление в образовании |
|---------|--------------------------|
| **Претензия** | Ученики недовольны учителями, учителя – программой, родители – школой. Вместо сотрудничества – взаимные обвинения. |
| **Неудовлетворённость** | Постоянная гонка за баллами, оценками, рейтингами. «Мало», «плохо», «надо лучше» — основной лейтмотив. |
| **Лживость** | Списывание, приписки, «красивые» отчёты без реальных знаний. |
| **Страх** | Страх экзаменов, страх ошибки, страх быть неуспешным. Парализует творчество. |
| **Гордыня** | Уверенность в своей правоте, нежелание учиться у других. |
| **Доминация** | Авторитарный стиль преподавания, подавление инициативы. |
| **Противоречивость** | Разные предметы учат разным картинам мира, нет целостности. |
