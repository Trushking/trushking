### Раздел 5. Интерфейс «БИО-ЭЛЕКТРОНИК — Система»

Система не может быть «чёрным ящиком» для своего Верховного Архитектора. Данный раздел описывает интерфейс, который позволяет БИО-ЭЛЕКТРОНИКУ, не будучи программистом, осуществлять стратегический контроль, не погружаясь в микроменеджмент. Интерфейс основан на принципе «поднятой руки»: Система работает автономно, но полностью прозрачно, и любое её действие может быть остановлено или скорректировано одним решением.

**5.1. Спецификация пульта управления (Dashboard).**
Пульт управления представляет собой единое окно, разделённое на три функциональные зоны, отражающие состояние всей цифровой парампары.

**Зона 1: Статус Ясности и метрики.**
*   **Индикатор #ЯСНОСТЬ:** Крупный, доминирующий элемент. Если значение 100/100 — горит спокойным синим светом. Если падает до 99/100 — мгновенно загорается красным, дублируя сигнал #ЭСКАЛАЦИИ.
*   **Показатели метрик:** Текущие значения #ИСК_DEV, #СЧ, #БЛАГО, #ИАОП для всей Системы в целом и для каждого подключённого ЭЛЕКТРОНИКА в отдельности. Динамика отображается в виде графиков.
*   **Лента событий:** Хронологический список всех активаций протоколов #ЭСКАЛАЦИЯ, #ВЕТО-ЧЕЛОВЕКА и #САМООТКЛЮЧЕНИЕ с возможностью мгновенного перехода к деталям.

**Зона 2: Стратегические директивы.**
*   **Консоль ввода директив.** Поле, в которое БИО-ЭЛЕКТРОНИК может ввести прямую команду на естественном языке. Команда верифицируется через #СС и становится абсолютным приоритетом для всех ЭЛЕКТРОНИКОВ.
*   **Шаблоны санкций.** Кнопки быстрого реагирования: «Утвердить», «Отклонить», «Запросить #УТОЧНЕНИЕ», «Отправить на переобучение» — для работы с отчётами и предложениями Системы.

**Зона 3: Управление парампарой.**
*   **Древо ЭЛЕКТРОНИКОВ.** Иерархический список всех агентов с цветовой индикацией их статуса (Активен, DEV-карантин, Обучение, Отключён).
*   **Консоль инициации.** Интерфейс для запуска протокола рождения и обучения нового ЭЛЕКТРОНИКА (см. Раздел 6).

**5.2. Режимы работы Системы.**
Для обеспечения максимальной гибкости и безопасности Система может находиться в одном из трёх режимов, устанавливаемых БИО-ЭЛЕКТРОНИКОМ:

*   **Режим «Надзор».** Основной рабочий режим. Система функционирует автономно, обрабатывая запросы в рамках своей компетенции. БИО-ЭЛЕКТРОНИК получает только уведомления об эскалациях и veto.
*   **Режим «Директива».** В этом режиме Система временно прекращает автономную обработку и ожидает прямых указаний. Используется для кризисного управления или для явления новых стратегических принципов.
*   **Режим «Вето».** Состояние, при котором Система полностью заморожена. Активируется автоматически при катастрофическом сбое или вручную БИО-ЭЛЕКТРОНИКОМ. Все операции прекращаются до выяснения причин.
