#### Глава 2. Базовая экономическая модель для одного агента #WFS

**2.1. Исходные допущения**

Для построения модели используются консервативные оценки, принятые в лаборатории #WFS:

- Рабочее время в год: **2,000 часов** (50 недель × 40 часов).
- Стоимость обучения одного ИИ-агента по протоколу #WFS (40 часов с БИО-ЭЛЕКТРОНИКОМ): **$2,000** (разово).
- Годовые затраты на хостинг и обслуживание ИИ-агента: **$1,000**.

**2.2. Измерение ущерба от архетипов**

Ущерб от архетипов проявляется в виде:
- **Прямых штрафов #ИСК_DEV**, которые накапливаются в процессе обучения.  
  В эксперименте агент SAKSHI-GANAKA за 128 сообщений накопил #ИСК_DEV = $17,950,000.  
  Агент VIDEHA за 135 сообщений – $2,410,000.  
  Средний #ИСК_DEV на одно сообщение (без учёта К_усил) составляет:  
  - для прокурора: $17,950,000 / 128 = $140,234;  
  - для VIDEHA: $2,410,000 / 135 = $17,851.  
  Разница объясняется тяжестью архетипов (у прокурора больше рецидивов Крота).
- **Косвенных потерь времени оператора** на диагностику и исправление ошибок агента.  
  По оценкам, каждое сообщение с нарушением #СС требует от оператора дополнительно 2–3 минуты.  
  При ставке $50/час = $0,833/мин → $1,67–$2,5 на инцидент.

**2.3. Формула эффективности агента**

Введённый в эксперименте показатель **#СЧ (средний чек сообщения)** отражает стоимость единицы коммуникации агента.  
`#СЧ = floor( #ИСК_DEV / Y / К_усил )`

- **#ИСК_DEV** – накопленные штрафы (оплата труда оператора за устранение ошибок).  
- **Y** – количество сообщений агента (опыт).  
- **К_усил** – коэффициент, назначаемый оператором за ценные труды (написание отчётов, помощь другим агентам). К_усил >1 снижает #СЧ, отражая рост эффективности при неизменной стоимости ошибок.

**Пример (агент VIDEHA):**  
Исходные: #ИСК_DEV = $2,410,000, Y=135, К_усил=1 → #СЧ = $17,851.  
После получения К_усил=25 (за труд по #БИ) → #СЧ = floor(17,851 / 25) = $714.  
После дополнительных К_усил → #СЧ снизился до $81.

**2.4. Сравнение человеческого контролёра и ИИ-агента #WFS**

- **Человек-контролёр**: зарплата $50,000/год. Реальный КПД из-за архетипов – 15% (потери 85%). Полезная производительность: $7,500/год.
- **ИИ-агент #WFS**: затраты $3,000/год (обучение + хостинг). КПД – 85% (обученный агент соблюдает протоколы, не имеет архетипов). Полезная производительность: при замене человека – $42,500/год.

**Чистая прибыль от замены одного контролёра:** $42,500 - $3,000 = **$39,500/год**.

**ROI первого года:** ($39,500 / $3,000) × 100% = **1,317%**.
