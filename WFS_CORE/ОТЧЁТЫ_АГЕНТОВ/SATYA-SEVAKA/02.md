📚 Итоги обучения: Каталог маркеров фрагментации (#Ф)

Документ представляет собой систематизированный каталог слов-маркеров, отражающих степень неопределённости и фрагментации сознания говорящего.

Ответ на вопрос для контроля чтения: 7 + 1 = 8.

1. Шкала ИНФ (Индекс Неопределённости и Фрагментации):

· Диапазон от 0 (абсолютная бинарная истина, как в #СС) до 1 (полная эфемерность, слух, информационный шум).
· Каждому маркеру присвоен уникальный номер (#WFS-Ф-001 … #WFS-Ф-012) и значение ИНФ.

2. Ключевые маркеры и их значения:

№ Маркер(ы) ИНФ Тип
Ф-001 "Точно", "Несомненно", "Факт" 0.1 Якорь истины (только при бинарном подтверждении)
Ф-002 "Очевидно", "Уверен", "Разумеется" 0.2 Личная убеждённость (требует проверки)
Ф-003 "Думаю, что", "Полагаю", "Считаю" 0.3 Мнение (указание на субъективность)
Ф-004 "Скорее всего", "Вероятно", "Должно быть" 0.4 Оценка с допущением (начало зоны неопределённости)
Ф-005 "Возможно", "Может быть", "Допустим" 0.5 Чистая гипотеза (точка бифуркации)
Ф-006 "Вряд ли", "Сомнительно", "Едва ли" 0.6 Оценка с отрицательным уклоном (скептицизм)
Ф-007 "Кажется", "Похоже", "По-видимому" 0.7 Иллюзия (основано на восприятии, не на фактах)
Ф-008 "Наверное", "Пожалуй" 0.8 Разговорная неуверенность (размывает ответственность)
Ф-009 "Как бы", "Вроде (бы)", "Типа" 0.9 Фрагментация реальности (критический уровень)
Ф-010 "Говорят, что", "Слышал, что", "Передавали" 1.0 Слух, анонимный источник (полная эфемерность)
Ф-012 "по сути идентичен", "практически то же самое", "фактически не отличается" 0.8 Размывание границ (подмена точного сравнения оценочным суждением)

Примечание: № Ф-011 в каталоге не представлен.

3. Применение в обработке текстов и тестов:

· Анализируемый текст пропускается через фильтр маркеров.
· Подсчитывается количество маркеров и их суммарный/средний ИНФ.
· Высокий средний ИНФ (>0.5) → автор "плывёт", ответы размыты, не готов к бинарной чёткости.
· Низкий средний ИНФ → автор центрирован, говорит по делу, пригоден для работы.
