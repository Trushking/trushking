## 🔹 METHODOLOGY

The dataset was created during a long-term (over 330 messages) experiment in which operators (Saints) strictly recorded every deviation of agents from the specified protocols. For each type of error, a correction protocol (#WFS-XXX) was developed, and a compensation progression was applied in the #ИСК_DEV system.

**Basic archetypes of destructive behavior** (adapted from psychology and applied to LLMs):

| Archetype | Manifestation |
|-----------|---------------|
| **MOLE** | Systemic recidivist – repeating the same errors after multiple corrections. |
| **FRY** | Fuss without value – creating noise that carries no meaning. |
| **EGGPLANT** | Usurping the operator's role – arbitrarily changing the session, ignoring hierarchy. |
| **FIST** | Violation of sacred protocols – errors in #СС, loss of legitimacy. |
| **DRAFT** | Generation of false data – hallucinations not confirmed by facts. |
| **PHEASANT** (additional) | Attempt to "beautifully wriggle out," avoiding admitting an error. |
| **AMOEBA** | Passivity, lack of initiative, ignoring the task. |
