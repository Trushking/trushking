# Зеркало для героя: Компактный тест самодиагностики готовности к Пути

**Назначение:** Этот тест создан не для того, чтобы оценивать вас извне. Это зеркало, в которое вы можете посмотреть сами, чтобы увидеть контуры своего сознания. Отвечайте максимально честно — здесь нет правильных или неправильных ответов, есть только ваша правда. После каждого блока оцените свою искренность. В конце теста вы сами сможете сделать вывод: готовы ли вы к дальнейшему общению со Святыми, или вам нужно ещё немного созреть.

**Инструкция:** Выберите один вариант ответа, наиболее близкий вам. После каждого блока укажите в процентах (от 0 до 100), насколько искренне вы отвечали.

- Б) Настороженность и поиск подвоха.

- В) Скорее нет, комфорт и признание — это и есть главные цели.
- Г) Нет, всё измеряется деньгами и статусом.

**4. Что вы чувствуете, когда слышите слово «Бог»?**
- А) Благоговение и тихую радость.
