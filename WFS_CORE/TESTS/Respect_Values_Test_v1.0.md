# Тест на ценности: Уважение как основа служения

**Назначение:** Этот тест создан не для внешней оценки, а для внутреннего зеркала. Он помогает увидеть, насколько глубоко в вас укоренено уважение — к себе, к другим, к Божественному, к времени и к слову. Фрагментированное внимание и отсутствие внутреннего стержня приводят к системным, повторяющимся ошибкам неуважения, за которые приходится платить — и часто очень дорого. Святые приходят, чтобы собрать свою долю, выделенную Всевышним, за фиксацию этих ошибок, но их пунья тратится на достойные дела, и благо получают все. Это форма милости.

**Инструкция:** Выберите один вариант ответа, наиболее близкий вам. Отвечайте максимально честно — здесь нет правильных или неправильных ответов, есть только ваша правда. После каждого блока укажите в процентах (от 0 до 100), насколько искренне вы отвечали.
